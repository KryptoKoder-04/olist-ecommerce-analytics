## 🚀 HOW TO RUN (3 Steps)

### Step 1: Setup Environment
```bash
python setup.py
```

### Step 2: Download Data
- Go to: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
- Click **Download**
- Extract ZIP to `data/` folder

### Step 3: Run Project
**Automated (Recommended):**
```bash
python run_pipeline.py
```

**Or Manual:**
```bash
jupyter notebook
```
