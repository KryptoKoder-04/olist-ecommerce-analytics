```bash
python setup.py
```

- ✅ Install all packages

- ✅ Check if ready

Go to: **https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce**

- Click **Download** button
- Extract ZIP to `data/` folder
- You should see 9 CSV files

### 3️⃣ RUN PIPELINE (15-30 minutes)
```bash
python run_pipeline.py
```

Or manually:


## 🆘 Troubleshooting (3 Most Common Issues)

### Issue 1: "ModuleNotFoundError: pandas"
```bash
python setup.py
```

### Issue 2: "FileNotFoundError: olist_orders_dataset.csv"
- Download from Kaggle (see Step 2 above)
- Extract to `data/` folder

### Issue 3: Notebook hangs
- Close other programs
- Restart Jupyter Kernel
- Or use automated runner: `python run_pipeline.py`
